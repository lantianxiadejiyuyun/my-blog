# My Blog 项目规划

> 定位：个人博客 + 评论互动，技术栈 Vue3 + TypeScript + Vite
> 后端：推荐 Node.js（你熟悉）或后续升级 Spring Boot（配合学习计划）
> 代码仓库：https://github.com/lantianxiadejiyuyun/my-blog

---

## 当前状态（进度约 10%）

### 已完成
- 项目脚手架：Vue3 + TS + Vite + SCSS
- 基础布局：顶部导航栏 + 内容区 + 底部栏
- 路由：首页（/）和关于我（/aboutme）两个页面
- 首页 Hero 区域 + 文章卡片骨架（假数据 v-for 3 条）
- 关于我页面：头像、联系方式、技能树圆环展示
- 静态资源：社交图标 SVG、技能树图片、背景图

### 存在的问题
1. **目录名拼写错误**：`commpoents` 应该是 `components`，文件名 `TopBarCommponets` 应该是 `TopBarComponents`
2. **Pinia 装了没用**：package.json 有依赖但 main.ts 没引入，没有 store
3. **Three.js 装了没用**：无任何 3D 代码，建议移除或明确用途
4. **无响应式设计**：AboutMe 页面 min-width: 1000px，手机端直接挂
5. **无文章详情页**：点击文章卡片没地方跳
6. **无 404 页面**
7. **无 API 层**：没有 axios、没有 fetch 封装、没有后端对接
8. **index.html 标题还是 "Vite App"**
9. **假数据全是 Lorem ipsum**
10. **config.ts 导出的变量没人用**

#### 数据库设计参考

```sql
-- 文章表
CREATE TABLE articles (
  id INT PRIMARY KEY AUTO_INCREMENT,
  title VARCHAR(200) NOT NULL,
  content TEXT NOT NULL,
  summary VARCHAR(500),
  cover VARCHAR(500),
  category_id INT,
  status TINYINT DEFAULT 0 COMMENT '0-草稿 1-已发布',
  view_count INT DEFAULT 0,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

-- 分类表
CREATE TABLE categories (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(50) NOT NULL UNIQUE,
  description VARCHAR(200)
);

-- 标签表
CREATE TABLE tags (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(50) NOT NULL UNIQUE
);

-- 文章-标签关联表
CREATE TABLE article_tags (
  article_id INT,
  tag_id INT,
  PRIMARY KEY (article_id, tag_id)
);

-- 评论表
CREATE TABLE comments (
  id INT PRIMARY KEY AUTO_INCREMENT,
  article_id INT NOT NULL,
  nickname VARCHAR(50) NOT NULL,
  email VARCHAR(100),
  content TEXT NOT NULL,
  reply_to_id INT DEFAULT NULL COMMENT '回复哪条评论',
  status TINYINT DEFAULT 0 COMMENT '0-待审核 1-已通过',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 管理员表
CREATE TABLE users (
  id INT PRIMARY KEY AUTO_INCREMENT,
  username VARCHAR(50) NOT NULL UNIQUE,
  password VARCHAR(200) NOT NULL,
  avatar VARCHAR(500),
  bio TEXT
);
```

## 和学习计划怎么配合

这个博客项目可以和学习计划并行推进：

| 学习计划阶段 | 博客项目对应 |
|-------------|-------------|
| 第 1 周：Pinia | 博客第二阶段 Day 2（接入 Pinia） |
| 第 2-3 周：Spring Boot | 博客第三阶段后端（用 Spring Boot 写） |
| 第 3 周：Redis | 博客加上文章缓存 |
| 第 4 周：Docker 部署 | 博客第五阶段部署上线 |

**建议**：先专注把前端（第一、二阶段）做完，后端等学完 Spring Boot 再写。这样博客就是你的**第二个完整项目**，比进销存项目更个人化，面试时两个项目一搭配，非常能打。
